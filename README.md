## Run locally

1. Clone the repo
   ```sh
   git clone https://github.com/nicodemos234/NiBot.git
   ```
2. Navigate to project folder
   ```sh
   cd NiBot/
   ```
3. Install packages
   ```sh
   yarn #I prefer yarn, but if you want to use npm use: npm install
   ```
4. Start the project
   ```sh
   yarn dev #or npm run dev
   ```
